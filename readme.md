# viewx-ui
viewx-ui是uniapp生态，一款多端、轻量、易用的前端UI框架，组件不求多，而在求精。

# 关于
开发过程中，笔者发现很多UI框架的组件虽然丰富，但是大部分在实际的项目中并不会用到亦或是使用起来不太方便，
因为项目大多是个性化的需求，遂将项目开发常用的组件提炼出来， 贴合实际，封装成一个UI库，viewx-ui由此诞生。
view是视图的意思，x代表着无限可能。
